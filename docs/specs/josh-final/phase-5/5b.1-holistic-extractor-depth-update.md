## `HolisticExtractOutput` Extension

Locate the type definition found in grep 2. Replace the `TODO` stub comment with the actual fields:

```ts
export type HolisticExtractOutput = {
  coordinationPatches?: Array<{
    path: string;
    op: "set" | "add" | "remove";
    value: unknown;
  }>;
  coordinationSignalPatches?: Array<{
    path: string;
    op: "set" | "add" | "remove";
    value: unknown;
  }>;
  activityPatternsAdd?: Array<{
    activity_key: string;
    motive_weights: Record<string, number>;
  }>;

  // Added by Ticket 5b.1
  interestSignaturePatches?: Array<{
    domain: string;
    intensity: number; // 0..1
    confidence: number; // 0..1, must be >= 0.40 to be written
  }>;
  relationalContextPatch?: {
    life_stage_signal?: string | null;
    connection_motivation?: string | null;
    social_history_hint?: string | null;
  };

  needsFollowUp: boolean;
  coverageSummary: DimensionCoverageSummary;
};
```

Both new fields are optional (may be absent from any extraction run). Presence of `interestSignaturePatches` does not guarantee all three `relationalContextPatch` fields are set — partial patches are valid.

```
Additionally, extract the following if inferable from the conversation:

INTEREST_SIGNATURES:
Look for domains the user elaborates on spontaneously, returns to, or speaks about with specificity.
These are deeper interests beyond their stated activity preferences.
Examples: "urban infrastructure", "endurance sports", "cooking as craft", "live music discovery".
For each identified domain, estimate:
- intensity: how engaged or passionate they seem (0.0–1.0)
- confidence: how confident you are in the inference (0.0–1.0)
Only include signatures with confidence >= 0.40. Omit this field entirely if none qualify.

RELATIONAL_CONTEXT:
Infer from how the user describes their social situation, what they're hoping to get from JOSH,
or any life transition signals they mention.
- life_stage_signal: e.g. "new to city", "recently divorced", "empty nester", "just retired"
- connection_motivation: e.g. "rebuilding social circle", "boredom", "loneliness", "staying active"
```
